# Unofficial-Gemini
Gemini-Fan-Client  A fast and minimalist C++ client for Google Gemini.  Installation:      Download the ZIP file from the Releases section.      Extract the contents.      Keep 'KI.exe' and 'WebView2Loader.dll' in the same folder.      Run 'KI.exe'.  Requirements:      Windows 10/11      Microsoft Edge WebView2 Runtime
